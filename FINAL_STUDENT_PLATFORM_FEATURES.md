# Explorisity Final Student Platform Feature Layer

This version keeps Explorisity's original school discovery, rankings, communities, social features, messaging, and comparison features.

The new features are added as secondary layers, not as the first thing users see.

## Added

1. Chance Me admissions discovery
2. Student Journey timeline
3. Optional academic bio fields
4. Test score display options for SAT, ACT, MCAT, LSAT, GMAT
5. Country and demographic profile fields
6. Rankings tied to student activity
7. Comparison remains a small supporting feature

## Suggested product hierarchy

Primary:
Discover, Schools, Rankings, Communities, Search

Secondary:
Chance Me, Compare, Messages, Saved, Profile, Settings

## Privacy

Academic scores, GPA, country, and demographic background should be optional and user controlled.
