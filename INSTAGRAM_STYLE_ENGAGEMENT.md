
# Instagram Style Engagement Updates

## Comments
Comments now conceptually support:
- username display
- avatar display
- clickable profile links

## Likes
Post like counts are hidden/minimal by default.
Users can click the likes area to open:
- modal
- drawer
- likes page

and see:
- usernames
- avatars
- profile links

## Suggested UX
Liked by:
- username
- and others

Clicking opens full list.
